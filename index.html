📋 Step 1 — Google Sheet setup (do this once)

  1. Go to sheets.google.com → create a new blank sheet
  2. Click Extensions → Apps Script
  3. Delete all existing code → paste the script from the box below → click 💾 Save
  4. Click Deploy → New deployment
  5. Click the gear ⚙ next to "Type" → select Web app
  6. Set Execute as: Me
  7. Set Who has access: Anyone
  8. Click Deploy → approve all permissions → copy the Web app URL
  9. Paste it in the field below → click Save & embed URL
Step 2 — Paste your Web App URL here

⚠️ This is the critical step. The URL must be saved here so that every staff member's phone sends their prediction directly to your Google Sheet.

✅ URL saved! Now do this final step:
1. Download the updated index.html file shown below
2. Go to your GitHub repository
3. Click on index.html → click the ✏️ pencil (Edit) icon → delete all content → paste the new content → click Commit changes
